Projeto IFB Mobile

Aplicativo desenvolvido durante pelos alunos do projeto LabApp.

O projeto tem a finalidade de ser o cartão postal da instituição IFB.

Apresentando ao público as notícias do instituto;

Quais cursos a instituição têm;

Detalhes das siglas para maior entendimento dos setores do campus;

E a parte para ajuda. No qual as principais duvidas são respondidas.
